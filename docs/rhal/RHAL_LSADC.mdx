# RHAL_LSADC

RHAL_LSADC api


## Function declarations and inline functions

### RHAL_LSADC_SetVoltageMode
```bash
DTV_STATUS_T RHAL_LSADC_SetVoltageMode(HAL_LSADC_NAME name, UINT8 voltageMode);
```
Set mode for a lsadc port
<ParamTable
  items={[
    { name: 'name', type: 'HAL_LSADC_NAME', desc: 'Specified lsadc pin name' },
    { name: 'voltageMode', type: 'UINT8', desc: 'voltage mode of lsadc' },
  ]}
/>
:::tip[voltageMode]

value = 1, set lsadc voltage mode<br />
value = 0, set lsadc current mode

:::


### RHAL_LSADC_SetActive
```bash
DTV_STATUS_T RHAL_LSADC_SetActive(HAL_LSADC_NAME name, UINT8 active);
```

<ParamTable
  items={[

    { name: 'active', type: 'UINT8', desc: 'set lsadc active or de-active' },
  ]}
/>
:::tip[active]

value = 0, set lsadc de-active<br />
value = 1, set lsadc active
