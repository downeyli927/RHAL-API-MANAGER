RHAL_SMC_Read​

SMC_STS RHAL_SMC_Read(void*pSmc, UINT8*pBuff, UINT32 Len, UINT32*pRxLen, UINT32 Timeout);

Read Data from Smart card (bypass protocol)

Parameters
pSmcvoid*: handle of a Smart Card IFD
pBuffUINT8*: read data buffer
LenUINT32: length of bytes to be read
pRxLenUINT32: number of readed bytes
TimeoutUINT32: wait time (in ms)

RHAL_SMC_Write​

SMC_STS RHAL_SMC_Write(void*pSmc, UINT8*pBuff, UINT32 Len);

Write Data to Smart card (bypass protocol)

Parameters
pSmcvoid*: handle of a Smart Card IFD
pBuffUINT8*: data to be written
LenUINT32: length of buff to be written

RHAL_SMC_Transfer​

SMC_STS RHAL_SMC_Transfer(void*pSmc, UINT8*pCAPDU, UINT32 APDULen, UINT8*pRAPDU, UINT32*pRAPDULen, UINT32 Timeout);

Do APDU layer data exchange

Parameters
pSmcvoid*: handle of a Smart Card IFD
pCAPDUUINT8*: C-APDU to be sent
APDULenUINT32: Size of C-APDU
pRAPDUUINT8*: R-APDU buffer
pRAPDULenUINT32*: (In) size of buffer to restire R-APDU / (OUT)size of received R-APDU
TimeoutUINT32: command timeout
