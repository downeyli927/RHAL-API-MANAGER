## RHAL_IS_IR_EVENT
```bash
SINT32 RHAL_IS_IR_EVENT(const CHAR*device_name);
```
<ParamTable
  items={[
    { name: 'device_name', type: 'const CHAR*', desc: 'device name' },
  ]}
/>
:::tip[Tips]

If device name is ir device, the return value is 1.
If device name is not ir device, the return value is 0.

## RHAL_OPEN_ALL_INPUT_DEVICES
```bash
DTV_STATUS_T RHAL_OPEN_ALL_INPUT_DEVICES(void);
```
Open all input devices
:::tip[Tips]

 If the function succeeds, the return value is OK.
 If the function fails, the return value is NOT_OK.
 If the function no receive data, the return value is NO_DATA_RECEIVED.

:::
