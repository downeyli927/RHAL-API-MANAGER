# RHAL_CEC_ADP

RHAL_CEC_ADP api

## RHAL_CEC_ADP_Open
```bash
HAL_CEC_STS RHAL_CEC_ADP_Open(void);
```
Open a RTK CEC Adapter

## RHAL_CEC_ADP_Close
```bash
HAL_CEC_STS RHAL_CEC_ADP_Close(void);
```
Close CEC ADP

## RHAL_CEC_ADP_SendMessage
```bash
HAL_CEC_STS RHAL_CEC_ADP_SendMessage(UINT8*pMsg, UINT8 Len);
```
